#  Netflix Data Analysis & Dashboard Project

## Overview

This project focuses on performing **Exploratory Data Analysis (EDA), SQL-based business intelligence, and interactive dashboarding** on a Netflix dataset containing information about movies and TV shows.

The goal was to extract meaningful insights, analyze trends, and build an **interactive Power BI dashboard** for better decision-making.

---

## Objectives

* Perform Exploratory Data Analysis (EDA)
* Answer business questions using SQL
* Identify patterns and trends in content
* Build an interactive dashboard using Power BI
* Generate actionable insights from data

---

## Tools Used

* **Python (Pandas, Matplotlib, Seaborn)**
* **SQL (MySQL)**
* **Power BI**
* **GitHub**

---

## Dataset Description

The dataset includes the following attributes:

* Content type (Movie / TV Show)
* Title
* Director
* Cast
* Country
* Date added
* Release year
* Rating
* Duration
* Genre
* Description

---

## Analysis Performed

### Exploratory Data Analysis (EDA)

* Analyzed content distribution (Movies vs TV Shows)
* Identified most common ratings
* Studied release year trends
* Examined content age and duration

---

### SQL Business Analysis

* Top countries producing content
* Monthly and yearly content trends
* Most frequent ratings
* Content type distribution
* Average duration analysis

---

### Key KPIs Implemented

* Content Growth Rate
* Movie-to-TV Show Ratio
* Content Freshness Score
* Mature Content Share
* Genre Diversity Index

---

### Deep-Dive Analysis

* Top contributing countries
* Yearly content growth trend
* Most popular genres
* Average movie duration (~100 minutes)
* Movies vs TV Shows trend over time

---

## Dashboard (Power BI)

An **interactive dashboard** was created to visualize insights:

* KPI Cards (Total Shows, Movies, TV Shows)
* Content distribution (Donut Chart)
* Content trend over time (Line Chart)
* Ratings analysis (Bar Chart)
* Global content distribution (Map)
* Filters for dynamic interaction
